Adds filters, sorting, and Unsubscribe button to the vanilla Mod Selector screen.

[b]What it adds:[/b]
Extends Project Zomboid's existing mod list with quality-of-life features - nothing is replaced.
[list]
[*] Filter by: [b]All[/b] / [b]Enabled[/b] / [b]Disabled[/b] / [b]Java[/b] / [b]New[/b]
[*] Sort by: [b]Name[/b] or [b]Date[/b]
[*] Unsubscribe Workshop mods directly from the in-game list
[*] Workshop mods you unsubscribed this session are hidden without restarting the game
[*] Shows each Java mod's load status next to its JAR path - [b]loaded[/b] / [b]blocked[/b], [b]allow[/b] / [b]deny[/b], [b]persisted[/b] / [b]session[/b] - color-coded (green = good, red = bad) so you can see at a glance which JARs ZombieBuddy let through
[/list]

[h1]FAQ[/h1]
[list]
    [*][b]Does it work on multiplayer?[/b]
    Yes - it only affects the client-side mod selector UI.

    [*][b]Save compatibility?[/b]
    Safe to add or remove on existing saves.

    [*][b]Supported game versions?[/b]
    B42.12+
[/list]
